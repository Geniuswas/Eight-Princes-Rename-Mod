# Eight-Princes-Rename-Mod

Renames characters from TOTAL WAR: THREE KINGDOMS Eight Princes DLC into new historical characters (with full localisation and itemisation support) that can be used in vanilla 190CE campaign.

Steam Workshop link: https://steamcommunity.com/sharedfiles/filedetails/?id=1897448383

This is an experimental version made for the finished game version of TW3K with IVoltz. 
Place .pack file into Total War THREE KINGDOMS/data folder and load with the official mod launcher.
